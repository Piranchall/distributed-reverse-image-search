# Distributed Reverse Image Search Engine

A distributed reverse image search engine built using parallel feature 
extraction, LSH indexing, and approximate nearest neighbor search.


## Dataset
MIRFLICKR-1M — [kaggle.com/datasets/sohangundoju/mirflickr-1m](https://www.kaggle.com/datasets/sohangundoju/mirflickr-1m)

## Pre-extracted Feature Files
Download from Kaggle: [mirflickr-1m-features]
(https://www.kaggle.com/datasets/piranchalghai/pdc-micflickr-1m-features)
